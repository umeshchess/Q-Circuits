# Q-Circuits
My Quantum Computing Journey

This repository documents my exploration into Quantum Logic and Circuits. As an Electrical and Electronics Engineering student, I am interested in how quantum gates represent a fundamental shift from classical CMOS logic.

## 🛠 Tools Used
* **Simulator:** [Quirk](https://algassert.com/quirk)
* **Environment:** Fedora Linux (Hyprland WM)
* **Concepts:** Dirac Notation, Bra-ket Algebra, Bloch Sphere analysis.

## 🧪 Experiments included

### 1. 4-Qubit GHZ State
A demonstration of maximal entanglement across four qubits.
* **Logic:** $H \to CNOT \to CNOT \to CNOT$
* **State:** $\frac{|0000\rangle + |1111\rangle}{\sqrt{2}}$
* **Key Learning:** Shared uncertainty and non-locality.

### 2. Bell States
Exploring the four fundamental entangled states for 2 qubits.
